# 5.5. PostgreSQL 설치

데이터베이스에 연결하기 위해 postgreSQL 프로그램을 설치한다. 다음 링크에 접속해 중앙에 있는 Download 버튼을 클릭한다.

[https://www.postgresql.org/](https://www.postgresql.org/)

<figure><img src="https://lh7-us.googleusercontent.com/Ubo7PnNiFGrKJtq9mZ1Zu6PVhssO1tccRZyX31nfMl_pu1CHksUuSu5s_SrmoFlCrz_51Hs5xvhE0--3Rfgum5Wvz2EEokg5z_vkpaIy-hYMHxDgyWwELoBEcUOs0pr3mGe0ahA0wfrJHf87j_r5Xqg" alt=""><figcaption><p>PostgreSQL 홈페이지</p></figcaption></figure>

Downloads 화면에서 원하는 OS를 선택한다. 여기서는 Mac에서 작업을 진행하고 있으므로 macOS를 선택한다.

<figure><img src="https://lh7-us.googleusercontent.com/svnKKkyz0RpLcaNdOo3IYDR4xB7aKkxYQ6L5F_Co0q3-JCzm7VWJOjnVkUF-Rh3S9wzUtnRfJkmB4vNXnkR8Haxj0X8mkvUQ31L17peywWGFNPwmQs0MvO6XWKbWRjZPk7q7EA2ZD2WG2dDfTWtb8-U" alt=""><figcaption><p>PostgreSQL 설치 OS 선택</p></figcaption></figure>

macOS packages 화면에서 상단의 Download the installer 화면을 클릭한다.

<figure><img src="https://lh7-us.googleusercontent.com/QwPeHZ2r44kyAXKgB_MkHy7ALBxHhAQzBhBRzhT9kkkiYdPdu3Y8a5XEzeIU9CaxFY3slWDFQXMl51bdt0zvlI69PKMZmVqHZhQDWrdTMsTV1Wpn73BlcfQF1qkm-WmjOY3vaPbfJpi9jEIWeTAhG2o" alt=""><figcaption><p>PostgreSQL macOS 다운로드 선택</p></figcaption></figure>

설치버전 선택 화면에서 원하는 버전의 설치파일을 다운로드한다. 이 문서에서는 15.4 버전의 Mac OS X 버전을 설치한다.

<figure><img src="https://lh7-us.googleusercontent.com/lREf-FdhHG_V1SfODDt8dt_Y1nE7zAIT4mSbiBZeBi_B9dCYjReR-vhGAou3QPZNhFR3vsXBsjmhVoVjtQI3fvnFdxF0Z7_Wt9f4zY99DvldHGabujXCS6JG99ErRxwQPns_rXIfPDTq0WWB3Pt2CDM" alt=""><figcaption><p>PostgreSQL 설치버전 선택 화면</p></figcaption></figure>

설치파일 실행 후 postgresql-15.4.1-osx.app 애플리케이션을 두번 클릭해 애플리케이션 설치를 시작한다.

<figure><img src="https://lh7-us.googleusercontent.com/vEaeOrFAqhNYbxtlViHHvyNx3kHIS8HjtzaeTBk4NdJ76FWlPzWA89DisKA7hsY_OqhQTBAZfUMEVyNup8dAa9w4O-c5junDePAr4ofRAEE-vKv3tOcB134QrHeZxGovED8fjRK2LC4o0Y3e49cnS2s" alt=""><figcaption><p>PostgreSQL 설치파일</p></figcaption></figure>

Setup 화면에서 Next 버튼을 클릭한다.

<figure><img src="https://lh7-us.googleusercontent.com/ZkzofJ2B7YwGk3nPvAaFD9AGQMcMotARcpk49BL2t7khAgZzW5g-zxSXu-p84NY21iNI5qCMINsWb8M2nB1Fe3HHpXVmdCPyWDvqYEp5Bhs3iQP8moLZ5w81Te-V6sKsaLneiAZ8EoyJwLPaVvD1DtU" alt="" width="563"><figcaption><p>PostgreSQL Setup</p></figcaption></figure>

Installation Directory 화면에서 설치위치를 지정 후 Next 버튼을 클릭한다.

<figure><img src="https://lh7-us.googleusercontent.com/CFCL2Auvb7atX-SskQEF-RxMKOXZ6aJtp4m4O4Dl5a0D2kGElMI6etaiU3lonIN2XHVR0xDu5Etel4rxk3QPtyvgeGEDVwjjo-gNwMk__jRzJ9ostYKJAbvbE3Nr7HldwVACYNrXBoeO9svDcD8VIfM" alt="" width="563"><figcaption><p>설치위치 설정</p></figcaption></figure>

설치 컴포넌트를 선택 후 Next 버튼을 클릭한다. 여기서는 전체 파일 선택 후 설치를 진행한다.

<figure><img src="https://lh7-us.googleusercontent.com/yGMFeg0WacUCNq7K8xI41Vj4tRciwq_iFum1Co1HewrDeWxBCx6hnaYDDQHAG1bVIyrDKWDJHu1kaFr_xY9fX8r-ABEIyaXAPX3GaIqA1ev7LTCrIH4dyA3RUkEr5OwUGSfu5AGZ0zj4DVmgVKWYS0s" alt="" width="563"><figcaption><p>설치 Components 선택</p></figcaption></figure>

Password 화면에서 데이터베이스 슈퍼유저의 비밀번호를 입력 후 Next 버튼을 클릭한다.

<figure><img src="https://lh7-us.googleusercontent.com/5EBiGIrWYka_4KgP7qbirFO30ZElsTW7-foupqE2nv94AU446R5NLok8-r01-z2eU2f1JaG9hbvNIkRF5Hd3mP3gLpJUiLyUnKZ83QEQ4Tdj17E8LN7y-LG_6kblKHR9BpJQKf08JKLqxts8gHTvM50" alt="" width="563"><figcaption><p>비밀번호 입력</p></figcaption></figure>

Port 입력 화면에서 포트번호를 5432로 입력 후 Next 버튼을 클릭한다.

<figure><img src="https://lh7-us.googleusercontent.com/HmrWaNw9bDotYkt-fQ_pTwrqEcgS-_RijQpXM86C0GLiYrTpsD91arEaEJiAq9vcXKhEnA4IQcvv3cbAAMW3n2r3Q67iMwRsaNEPyoHGxRDAU54n9k2xuPNGYStfVeAiivnAKPxzxV6xBgx9z3CuHg8" alt="" width="563"><figcaption><p>포트번호 입력</p></figcaption></figure>

데이터베이스 클러스터 위치선택 화면에서 데이터베이스 클러스터가 설치될 위치를 ko\_KR.eucKR로 설정 후 Next 버튼을 클릭한다.

<figure><img src="https://lh7-us.googleusercontent.com/i_JH2kqXM3xbB5VK7mtXqDqByoctWN6jTb4Uo82kHaa_n8NFOrUCvj4f0s3GvUt0ZSVSBPGlS21ouSmoQT0ndCe29oia0q7aLyFsE2lxxGpXnOdKvvYiC4YBuEb1pF7pBSJe-FLOfPDIUcXysMU6lLI" alt="" width="563"><figcaption><p>데이터베이스 위치 설정</p></figcaption></figure>

설치 관련된 설정이 끝나면 설치정보 요약 화면이 나타난다. 설치정보가 올바르게 되었다면 Next 버튼을 클릭해 설치를 진행한다.

<figure><img src="https://lh7-us.googleusercontent.com/t2uh4gRKMQdDYAdiIFoBiMyj-skoWjl9YUZrFH5IvpMZmxhBtEvNEm5OKSHbrJ3X3oWetKD1_D5oBOoUMm7VVGoe1a79Vs2b3R4khKQwpV-rzbRN0rtAbyVOkYbW_3iid9jmS8Y5arry_c74e7y29Ps" alt="" width="563"><figcaption><p>설치정보 요약</p></figcaption></figure>

설치준비 화면이 나타나면 Next 버튼을 클릭해 설치를 진행한다.

<figure><img src="https://lh7-us.googleusercontent.com/KF9UIrbIeXKv9Cwkx7Br6DMvRS1iO_PRG7ahy5q2pOIGu7z8A8tEHREK-jvu_QV4xlNQ1ihKxccrzTTPQkly3fIhSDlGToSGuSCNfzaPvdvL9XLGc2Be-mrumiEYYci0PtmLxg7QwQKZMVv9DlGhJCw" alt="" width="563"><figcaption><p>설치설정 완료</p></figcaption></figure>

설치가 완료되면 Finish 버튼을 클릭해 설치를 완료한다.

<figure><img src="https://lh7-us.googleusercontent.com/Ajmcs-_t28MAUerduxc1RlV6--2w4cdSxxF40mLj1Zlma-YaxmV4qNxUnzz1FjYzhFZ1_9aCCwa5UWFP0PAyLnxtPRVh1ktwpz0TLw_EJDTjcx5JBJaFDXXx4GxRnWdyTd7Kr_vdmHVrN3kL3Bkgv2I" alt="" width="563"><figcaption><p>설치완료 화면</p></figcaption></figure>

이후 설치가 완료되면 컴퓨터에 설치된 pgAdmin 4 를 클릭해 실행한다.
